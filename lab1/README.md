# Анализ данных о продуктах питания

## Описание данных
Данные содержат информацию о различных продуктах питания, включая показатели:
- `название`: название продукта (строка)
- `категория`: категория продукта (строка)
- `калорийность`: калории на 100 г (число)
- `вес`: вес продукта в граммах (число)
- `белки`: содержание белков на 100 г (число)
- `жиры`: содержание жиров на 100 г (число)

## Используемые библиотеки
- pandas
- matplotlib
- seaborn

## Методы предобработки данных
- Проверка типов данных
- Обнаружение и обработка пропущенных значений
- Преобразование типов данных
- Построение графиков для анализа распределения и взаимосвязи переменных
